# WeChat2Telecom (微信来电系统同步工具)
WeChat2Telecom 是一个轻量级的安卓辅助工具，旨在将微信语音/视频通话通过 Android Telecom 框架桥接为系统级电话提醒。

## 🚀 核心功能
同步振铃：当微信收到来电时，触发系统原生通话界面，使 Android Wear 手表、华为/小米手环以及 Windows (通过“手机连接”) 同步响铃并显示联系人。

生命周期同步：微信挂断后，系统级通话提醒会在 1.5 秒内自动清理，确保不产生冗余提醒。

极简设计：无广告、无多余权限、仅专注于通知转换。

## 🛠 关键权限配置 (必读)
由于安卓系统（尤其是 MIUI/HyperOS/OriginOS）对后台行为限制极严，请务必手动完成以下设置，否则服务可能无法正常工作：

通知监听权限：安装后首次启动，请按提示开启“通知监听服务”。

无障碍浮窗监听：建议开启“微信来电浮窗监听”。它只读取微信来电浮窗状态，不点击、不接听、不控制微信；用于在通知文本不稳定时判断来电和挂断。

默认通话账户：进入 系统设置 -> 应用管理 -> 默认应用 -> 电话（或在拨号盘设置中），将 WeChat Bridge 设为可选的通话账户。

后台弹出界面：在“应用详情 -> 权限管理”中，将 后台弹出界面 设为“始终允许”。

省电策略：在“应用详情 -> 电池管理”中，设置为 “无限制” 或“不优化电池使用”。

开机自启：模块会在开机、解锁或应用更新后重新初始化 Telecom 账户和通知监听组件；部分系统仍需在自启动/后台管理里手动允许。

## 📖 使用说明
运行模式：本版本只监听微信通知并同步响铃，不 Hook 微信进程，不注入触摸事件，也不控制微信接听或挂断。

当微信来电时，模块会桥接为系统来电，让手表、手环或 Windows 手机连接同步响铃；在外部设备点击接听、拒绝或挂断，只会关闭本地系统提醒，不会影响微信。手机端接通后，本地系统来电也会自动结束。
